# GpuShark Telemetry

GpuShark is a lightweight console application designed for real-time system monitoring. It provides a direct readout of essential GPU and CPU sensor data in a simple text-based interface.

## Features

The tool monitors and displays the following telemetry via Direct Memory Access:

* **GPU Statistics**
    * Core Temperature
    * Hot Spot Temperature
    * VRAM Temperature
    * VRAM Usage (GB)
    * Fan Speed (RPM)

* **CPU Statistics**
    * Package Temperature

## Requirements and Usage

* **OS:** Windows 10 / 11
* **Privileges:** Requires Administrator rights to access hardware sensors correctly.

**How to run:**
1.  Download the executable.
2.  Right-click `GpuShark.exe` and select **Run as Administrator**.
